# 🏦 Bank Loan Analytics Dashboard

## 📌 Project Overview

This Power BI dashboard analyzes bank loan applications, approvals,
loan amounts, applicant profiles, and credit behavior.

The dashboard enables users to monitor loan performance,
analyze customer characteristics,
and gain business insights through interactive visualizations.



## 🎯 Objectives

- Analyze loan applications
- Monitor approval and rejection trends
- Compare customer education and employment status
- Evaluate CIBIL score impact
- Build an interactive Power BI dashboard


## 📂 Dataset

- Records: 4000+
- Columns: 12
- Domain: Banking


## 📈 KPIs

- Total Loan Applications
- Total Loan Amount
- Average Loan Amount
- Average Income
- Average CIBIL Score
- Approved Loans
- Rejected Loans
- Approval Rate


## 📊 Dashboard Features

- KPI Cards
- Loan Status Distribution
- Loan Amount by Education
- Loan Amount by Self Employed
- Average CIBIL Score Analysis
- Loan Applications by Loan Term
- Interactive Slicers


## 💡 Business Insights

- Loan Approval Rate is 62.22%.
- Approved applicants have significantly higher CIBIL scores.
- Loan amounts are nearly equal across education groups.
- Loan distribution is balanced between self-employed and salaried applicants.


## 🛠 Tools Used

- Power BI
- Power Query
- DAX
- Microsoft Excel
